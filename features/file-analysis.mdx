# File Analysis

Upload any document and Sage will read, analyze, and answer questions about it.

## Supported Files
- **PDF** — Textbooks, past questions, lecture slides
- **Images** — Photos of notes, diagrams, questions
- **Text files** — Any plain text document

## What You Can Do
- "Summarize this document"
- "Create quiz questions from this PDF"
- "Explain page 3 in simple terms"
- "What are the key points for my exam?"
