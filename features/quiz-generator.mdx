# Quiz Generator

Generate practice quizzes on any topic instantly.

## How It Works
1. Go to the **Quiz** tab
2. Enter a topic (e.g., "Organic Chemistry")
3. Choose number of questions (5-20)
4. Take the quiz with instant feedback
5. Review explanations for wrong answers
