# Troubleshooting

**Sage isn't responding**
- Check your internet connection
- Refresh the page
- Try again in a few seconds (the AI might be busy)

**Quiz won't generate**
- Make sure your topic is specific enough
- Try a different wording

**File upload fails**
- Maximum file size is 10MB
- Supported: PDF, PNG, JPG, TXT
- Try a smaller file or compress it

**Voice features not working**
- Voice requires Premium plan
- Make sure your browser allows microphone access
- Voice doesn't work on 2G connections

**Storage full error**
- StudyOps automatically cleans up old data
- Clear your browser cache if the issue persists
